# О программе

Программа для скачивания фильмов с помощью сайта The movie Database. Это уникальный в своем роде каталог фильмов 
и данных о них 

## make_own_db

Наш главный скрипт. Скачивает фильмы из базы данных the movie database.
Для запуска необходимо получить Апи-ключ этого проекта. Необходимо сделать следующее:

1. Зарегистрируйтесь на сайте the movie database [вот тут ](https://api.themoviedb.org)
2. Зайдите в раздел настройки и выберите меню вариант **API**
3. Подайте заявку и заберите ваш ключ.

Далее запускаем командой `python make_own_db.py` и вставляем наш API-ключ в терминал


## search_in_DB

Запускается командой `python search_in_DB.py`

Для запуска необходимо в терминале указать путь (абсолютный) к папке с фильмами

Ищет фильмы в Базе данных ( папке ) по названию и выдает все похожие результаты
Что то похожее делают другие кино-сервисы по типу "Кинопоиск"


## find_similar

Запускается командой `python find_similar.py`

Для запуска необходимо указать путь к папке с файлами
Сравнивает фильмы по рейтингу, сортирует их от низшего к высшему рейтингу и возвращает пользователю 

**Как скрипт сортирует фильмы?**

Все очень просто.
Рейтинг в понимании скрипта, это схожесть вашего фильма со сравниваемым.
Он берет ваш фильм с текущим (с которым идет сравнение) и смотрит:

- Если жанры одинаковые, то добавляет очки в общий рейтинг
- Если они одинаковы (например, оба 'Comedy'), то условие истинно, и соответственно, добавляет очки.

**Проще говоря, если параметры фильмов одинаковые, то скрипт добавляет в рейтинг очки,
и в конце сортирует фильмы по ним.** 

## hello_api_TMDB

Запускается командой `python hello_api_TMDB.py`

Для запуска необходим API ключ the movie database. 

Делает запрос к API the movie database для проверки API-ключа и выводит ответ пользователю
